🎯 What is this?

This website is an interactive browser for the awesomedata/awesome-public-datasets GitHub repository — a topic-centric list of high-quality open datasets for researchers, data scientists, students, and developers. Every dataset listed here is publicly accessible and free to use.

This is a community-driven project. The original repository is maintained by volunteers who continuously add, verify, and categorize datasets from across the web.

📂 What's inside?

The directory spans 33 topic categories including:

  • Science & Research — Biology, Chemistry, Physics, Neuroscience, EarthScience
  • Society & Economy — Economics, Finance, Government, Education, Healthcare
  • Technology — MachineLearning, ComputerNetworks, CyberSecurity, Software
  • Geospatial & Environment — GIS, Climate+Weather, Transportation
  • Culture & Media — Entertainment, Sports, Museums, NaturalLanguage

🔍 How to use this site

  • Browse by category using the left sidebar
  • Search across all datasets using the search bar at the top
  • Click any card to view the full description and access the dataset link
  • Use the Visit Dataset button in the popup to go directly to the data source

🌐 Source

All dataset metadata is sourced from the open-source GitHub repository awesomedata/awesome-public-datasets. This site is a read-only browser — it does not modify, store, or redistribute the underlying datasets.
